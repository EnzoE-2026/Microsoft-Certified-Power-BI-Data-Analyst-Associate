# Microsoft Certified Power BI Data Analyst Associate

[Microsoft Certified: Power BI Data Analyst Associate](https://learn.microsoft.com/en-us/credentials/certifications/data-analyst-associate/?practice-assessment-type=certification)

The PL‑300 (Microsoft Power BI Data Analyst) certification validates your ability to turn raw data into actionable insights using the full Power BI workflow. It covers preparing, modeling, visualizing, analyzing, and securely deploying data solutions. Candidates are expected to deliver meaningful business value through clear, domain‑informed analytics and intuitive visualizations.
